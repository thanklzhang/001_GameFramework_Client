HDRP and URP are supported

In order to use the asset in the HDRP/URP: 
 - You can simply use the Unity's deafult method: "Edit->Renderer Pipeline->Upgrade Materials"
 - Or you can import HDRP/URP_Upgrade unitypackage and replace the materials.

The deafult color space is Linear.
In order to use the asset in the Gamma color space, you need to import Gamma_path unitypackage

***
Important Note:
Soft Particles are supported only in the Built-In RP.
